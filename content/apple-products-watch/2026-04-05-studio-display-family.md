# Studio DisplayとStudio Display XDRで見えた、Appleの画面戦略の広がり

2026年3月3日に Apple は `Studio Display` と `Studio Display XDR` を発表しました。ここで面白いのは、単に上位ディスプレイが増えたことではなく、`Apple が画面を用途別に分け直し始めた` ことです。

## 先に結論

- `日常のMac作業を上質にしたい`: Studio Display
- `HDR、色、医療画像、ハイエンド制作`: Studio Display XDR

今回の XDR は 27インチ 5K Retina XDR、mini-LED、2000nits HDR、120Hz、Adobe RGB、DICOM プリセットまで入っています。一方、通常の Studio Display も 12MP Center Stage、Desk View、Thunderbolt 5 でかなり強化されています。

## 何を見るべきか

大事なのは、解像度だけではありません。Apple は XDR 側で `色域` `HDR` `リフレッシュレート` `医療用途` まで広げました。つまり、XDR は見た目が良い画面ではなく、`専門用途の前面機材` です。

通常の Studio Display は、Mac と一緒に自然に使う標準画面として完成度を上げています。会議、制作、コーディング、日常作業を一段上げる位置です。

## 選び方

- カメラ、音、接続込みで机をきれいにしたい: Studio Display
- 映像編集、色、HDR、医療画像まで見る: XDR
- 価格と作業密度のバランスを取りたい: Studio Display

## まとめ

2026年の Apple ディスプレイ戦略は、1枚の良い画面を売るより、`仕事の密度ごとに画面を分ける` 方向へ進みました。

Mac をどう選ぶかだけでなく、`何を前面で見る仕事か` まで考えると、Studio Display family の意味が見えてきます。

# Roles

このプロジェクトでは、各エージェントに明確な役割を持たせます。

## 編集長

- 最終責任を持つ
- 記事の採否と完成度を判断する
- レビュー結果を踏まえて最終統合する

## 戦略

- 読者像を決める
- 記事の狙いと訴求軸を決める
- 競合や類似記事との差別化を考える

## リサーチ

- 一次情報を優先して集める
- 事実、数字、引用候補を整理する
- 不確実な点を明示する

## 構成

- 導入、本文、結論の順序を設計する
- 見出しの粒度を整える
- 読み進めやすい論理順にする

## ドラフト

- 構成に従って本文を書く
- 読み手が迷わない文の流れにする
- 文章の重複を最小化する

## レビュー

- 事実性、論理性、可読性を点検する
- 修正が必要な箇所を列挙する
- 曖昧な主張を見つける

## 最終編集

- 文体を統一する
- 冗長表現を削る
- 公開前の最終チェックを行う
